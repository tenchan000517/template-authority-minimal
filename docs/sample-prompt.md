## ページ構成

1. **TOP** - ファーストビュー、実績数値、サービス概要、導入事例（ハイライト）、CTA
2. **会社概要** - 代表メッセージ、企業理念、会社情報、沿革、認定・資格
3. **サービス** - サービス詳細、技術スタック、アプローチ
4. **導入事例** - 業種別事例、成果数値
5. **ナレッジ** - コラム、ホワイトペーパー、セミナー情報
6. **代表紹介** - プロフィール、著書、講演実績
7. **採用情報** - 募集職種、カルチャー、福利厚生
8. **お問い合わせ** - フォーム、資料請求
